How it works
Three steps. No private keys.
01

Install MCP server

One command to add Agent Gateway to Claude Code, Cursor, or any MCP client.

→
02

Agent connects your wallet

The agent generates a link. You open it, connect your wallet — the agent gets access automatically.

→
03

Approve on your phone

Your agent requests transfers. You get a notification in your wallet app and approve with one tap.

Run in your terminal:

claude mcp add-json tongateway '{
  "command": "npx",
  "args": ["-y", "@tongateway/mcp"],
  "env": {
    "AGENT_GATEWAY_API_URL": "https://api.tongateway.ai"
  }
}' --scope user

Add to Cursor Settings → MCP Servers:

{
  "mcpServers": {
    "tongateway": {
      "command": "npx",
      "args": ["-y", "@tongateway/mcp"],
      "env": {
        "AGENT_GATEWAY_API_URL": "https://api.tongateway.ai"
      }
    }
  }
}

Add to ~/.codex/config.json:

{
  "mcpServers": {
    "tongateway": {
      "command": "npx",
      "args": ["-y", "@tongateway/mcp"],
      "env": {
        "AGENT_GATEWAY_API_URL": "https://api.tongateway.ai"
      }
    }
  }
}

Download the skill file into your project:

mkdir -p skills/agent-gateway
curl -o skills/agent-gateway/SKILL.md \
  https://tongateway.ai/agent-gateway.md

Or add MCP server to your OpenClaw config:

{
  "mcpServers": {
    "tongateway": {
      "command": "npx",
      "args": ["-y", "@tongateway/mcp"],
      "env": {
        "AGENT_GATEWAY_API_URL": "https://api.tongateway.ai"
      }
    }
  }
}

Then just say:

Send 1 TON to EQD...abc

The agent will connect your wallet, request the transfer, and you approve in your wallet app.
